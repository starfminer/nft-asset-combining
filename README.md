# `NFT Pipeline Toolbox`



A collection of Python utilities for generating, validating, and optimizing

ERC-721 NFT collections.



This repository is organized as a modular toolbox so individual components

can be reused across different NFT projects.



## Structure



### `generator`

Layered NFT asset generation with weighted rarity and ERC-721 compatible

metadata output.



### `image`

Image processing utilities such as batch resizing and optimization.



### `metadata`

Metadata validation tools for supply counts, trait consistency, and

collection integrity.



## Tech Stack



- Python 3

- Pillow (PIL)

- JSON
